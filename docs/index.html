Content

  1. Introduction to HTML5
  2. Introduction to CSS3
  3. Introduction to Bootstrap. The course deals with Bootstrap 4, but I have worked with Bootstrap 5; I had a lot of problems due to the differences between the two versions, but it was a good opportunity to deepen my knowledge on Bootstrap.
  4. JavaScript. You must have previous knowledge of a programming language. Some "advanced" topics are treated very well, in particular OOP.
  5. DOM manipulation. Handling events. Introduction to Ajax (HTTP basics, Ajax basics, Processing JSON). Dynamically loading content in web pages.

Final project (Restaurant site)

  1. https://alec58.github.io/CourseraClientWebDev/ProgettoRistorante/ristorante01/

    The first step deals with: semantic HTML, CSS basics, responsive layuot, fonts and icons, Bootstrap basics, Bootstrap grid system, focus on navbar component (the course (feb 2021) deals with Bootstrap 4, but I have worked with Bootstrap 5 beta 2), creation of the basic layout of the home page and navigation menu

  2. https://alec58.github.io/CourseraClientWebDev/ProgettoRistorante/ristorante02/

    The second step deals with: creation of the main content of the home page using Bootstrap, responsive images, insertion of Google maps, media query, "card" component

  3. https://alec58.github.io/CourseraClientWebDev/ProgettoRistorante/ristorante03/

    The third step deals with: creation of menuCategories.html and singleCategory.html pages. Lots of formatting

  4. https://alec58.github.io/CourseraClientWebDev/ProgettoRistorante/ristorante04/

    The fourth step deals mainly with JavaScript programming and AJAX. I have used AJAX with the help of jQuery. Main content of index.html page is now loaded dynamically from snpIndex.html file using AJAX request. NB: if you test the project "locally" you get CORS errors! You must use a web server (it's ok to serve pages through browser-sync)

  5. https://alec58.github.io/CourseraClientWebDev/ProgettoRistorante/ristorante05/

    Also the fifth step deals mainly with JavaScript programming and AJAX. We also move from a "traditional" site made up of many pages to a SPA (Single Page Application). In this step we make the code equivalent to the "menuCategories.html" page

  6. https://alec58.github.io/CourseraClientWebDev/ProgettoRistorante/ristorante06/

    Continue as in the previous step, this time creating the code equivalent to the "singleCategiry.html" page

  7. https://alec58.github.io/CourseraClientWebDev/ProgettoRistorante/ristorante07/

    Final step :-). Fix "active button" style. About, Awards & Specials pages created
